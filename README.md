# ⚙️如何编译项目？

一、全局安装 Yarn 包管理器

```bash
npm install -g yarn
```

二、拉取该项目，并安装依赖

```bash
yarn install
```

三、打包编译

```bash
npx webpack
```

四、打开`dist/index.html`运行游戏
